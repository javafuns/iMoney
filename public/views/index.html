Introduction:

iMoney application is composed of 3 services, which are listed and described below.

It is built on Node.js, packaged as a docker image, deployed and run in Google Container Engine.

Cost Management

Record your cost in Cost Management.

Cost Analytics:

Based on cost management service, we generate a consumption report with Bar, Line, Pie charts, go Cost Analytics.

Stock Inquiry:

Stock Inquiry service is embeded in Cost Analytics. You can query a stock's latest Last Trade Price.

Technologies: Yahoo Financial API and JQuery
